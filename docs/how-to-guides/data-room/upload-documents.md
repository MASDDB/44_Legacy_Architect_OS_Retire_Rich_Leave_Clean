# How to Upload Documents to Your Data Room

**Organize your business documents for M&A readiness.**

*Time required: 5-30 minutes (depending on number of files)*

---

## Before You Start

### What You'll Need

- Documents to upload (PDF, DOC, DOCX, XLS, XLSX, images)
- Files should be:
  - Under 50MB each
  - Named clearly (e.g., "2024_P&L_Statement.pdf")
  - Organized by type

### Recommended Preparation

1. **Gather documents** from various sources
2. **Rename files** with clear, descriptive names
3. **Remove personal/confidential info** not needed for business sale
4. **Organize into categories** matching Folder 0-10 structure

---

## Method 1: Upload Individual Files

### Step 1: Navigate to Data Room

1. Log in to Legacy Architect OS
2. Click **"Data Room"** in left sidebar
3. You'll see the Folder 0-10 structure

### Step 2: Select Target Folder

1. Click on the folder where document belongs
   - Example: Click **"Folder 2: Financial Information"** for P&L statements
2. The folder opens showing existing documents (if any)

### Step 3: Upload File

**Option A: Drag and Drop**
1. Open your file explorer/finder
2. Locate the file
3. Drag file into the upload area
4. Drop when you see "Drop file here"

**Option B: Click to Browse**
1. Click **"Upload Document"** button
2. Browse to select file
3. Click **"Open"**

### Step 4: Add Document Details (Optional)

After upload, you may be prompted to add:

**Document Name** (auto-filled from filename)
- Edit if needed for clarity
- Example: Change "scan001.pdf" to "Articles of Incorporation"

**Document Type** (select from dropdown)
- Corporate Document
- Financial Statement
- Contract
- Tax Return
- etc.

**Description** (optional)
- Brief note about the document
- Example: "Audited financial statements for fiscal year 2024"

**Date** (optional)
- Document date or period covered
- Example: "12/31/2024" for year-end financials

### Step 5: Save

1. Click **"Save"** or **"Upload"** button
2. Wait for upload to complete
   - Progress bar shows status
   - Green checkmark when done
3. Document appears in folder list

---

## Method 2: Upload Multiple Files

### Bulk Upload Process

1. Navigate to target folder
2. Click **"Upload Multiple"** button
3. Select multiple files:
   - **Windows**: Hold Ctrl and click files
   - **Mac**: Hold Cmd and click files
   - Or select all files in a folder
4. Click **"Open"**
5. All files upload simultaneously
   - Progress shown for each file
   - Can continue working while uploading

### Best Practices for Bulk Upload

- **Upload 10-20 files at a time** (not 100+)
- **Use consistent naming** for easier organization
- **Group by folder** - upload all Folder 2 docs together, then Folder 3, etc.
- **Check file sizes** - large files take longer

---

## Method 3: Import from Cloud Storage

If you have documents in Google Drive or Dropbox:

### Connect Cloud Storage First

See: [How to Connect Google Drive](connect-google-drive.md) or [How to Connect Dropbox](connect-dropbox.md)

### Import Files

1. Navigate to **Data Room** → **Cloud Storage**
2. Click **"Import from Google Drive"** or **"Import from Dropbox"**
3. Browse your cloud storage
4. Select files or folders to import
5. Map to appropriate Data Room folders
6. Click **"Import"**

**Advantages**:
- No re-uploading needed
- Faster for large numbers of files
- Maintains original file organization

## File Requirements

### Supported File Types

**Documents**:
- PDF (.pdf) - **Recommended for final documents**
- Microsoft Word (.doc, .docx)
- Microsoft Excel (.xls, .xlsx)
- Text files (.txt, .rtf)

**Images**:
- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)

**Other**:
- CSV (.csv) - for data
- ZIP (.zip) - for bulk uploads

### File Size Limits

- **Per file**: 50MB maximum
- **Total storage**: Varies by plan
  - Free: 1GB
  - Professional: 10GB
  - Enterprise: 100GB+

### File Naming Best Practices

**Good names**:
- `2024_Annual_P&L_Statement.pdf`
- `Customer_Contract_Acme_Corp.pdf`
- `Articles_of_Incorporation_2015.pdf`

**Bad names**:
- `scan001.pdf`
- `document.docx`
- `IMG_1234.jpg`

**Tips**:
- Include date or year
- Include document type
- Include company/client name (if applicable)
- Use underscores or hyphens (not spaces)
- Be specific and descriptive

### Use the Folder 0-10 Structure

Upload documents to the correct folder:

**Folder 0**: Executive Summary & Teaser
**Folder 1**: Corporate Documents
**Folder 2**: Financial Information
**Folder 3**: Tax Information
**Folder 4**: Operations
**Folder 5**: Sales & Marketing
**Folder 6**: Customers & Contracts
**Folder 7**: Human Resources
**Folder 8**: Technology & IP
**Folder 9**: Legal & Compliance
**Folder 10**: Real Estate & Assets

See: [Building Your First Data Room](../../tutorials/building-data-room.md) for detailed folder descriptions

### Create Subfolders (Optional)

Within each main folder, you can create subfolders:

1. Click on main folder
2. Click **"New Subfolder"** button
3. Name the subfolder
   - Example: In Folder 2, create "P&L Statements" and "Balance Sheets"
4. Upload documents to subfolders

## Managing Uploaded Documents

### View Documents

1. Navigate to folder
2. Click on document name to view
   - PDFs open in browser
   - Other files may download

### Download Documents

1. Hover over document
2. Click download icon
3. File downloads to your computer

### Replace/Update Documents

1. Hover over document
2. Click **"Replace"** or **"Upload New Version"**
3. Select new file
4. Old version is archived (not deleted)

### Delete Documents

1. Hover over document
2. Click delete/trash icon
3. Confirm deletion
4. Document moves to trash (can be restored for 30 days)

### Move Documents

1. Select document (checkbox)
2. Click **"Move"** button
3. Select destination folder
4. Click **"Move Here"**

## Troubleshooting

### Problem: Upload fails or times out

**Solutions**:
1. **Check file size** - Must be under 50MB
2. **Check internet connection** - Stable connection required
3. **Try smaller batches** - Upload 5-10 files at a time
4. **Compress large files** - Use ZIP or reduce PDF quality
5. **Try different browser** - Chrome recommended

### Problem: File won't open/preview

**Solutions**:
1. **Download and open locally** - Some formats don't preview
2. **Convert to PDF** - Most compatible format
3. **Check file isn't corrupted** - Try opening on your computer first

### Problem: Can't find uploaded document

**Solutions**:
1. **Check correct folder** - May have uploaded to wrong folder
2. **Use search** - Search by filename
3. **Check trash** - May have been accidentally deleted
4. **Refresh page** - Sometimes display lags

### Problem: "Storage limit exceeded"

**Solutions**:
1. **Delete unnecessary files** - Remove duplicates, old versions
2. **Compress files** - Reduce PDF sizes
3. **Upgrade plan** - Get more storage
4. **Use cloud storage import** - Doesn't count toward storage limit

## Best Practices

### Upload Strategy

**Week 1**: Critical documents (20% of total)
- Financial statements
- Corporate documents
- Top customer contracts

**Week 2-3**: Important documents (50% of total)
- All contracts
- HR documents
- Tax returns

**Week 4+**: Remaining documents (30% of total)
- Marketing materials
- Process documentation
- Supporting documents

### Quality Over Quantity

**Better to have**:
- 50 well-organized, clearly named documents
- Complete key folders (1, 2, 6)
- Professional presentation

**Than**:
- 200 random, poorly named files
- Incomplete critical folders
- Messy organization

### Regular Maintenance

**Monthly**:
- Upload new documents as created
- Update changed documents
- Remove outdated documents

**Quarterly**:
- Review all documents for accuracy
- Check for missing documents
- Reorganize if needed

## Security & Privacy

### Who Can See Your Documents?

**Only you** can see your Data Room documents, unless:
- You invite team members (with appropriate permissions)
- You share Data Room with advisors/buyers (you control access)

### Document Encryption

- All documents encrypted in transit (HTTPS)
- All documents encrypted at rest (AES-256)
- Secure storage on enterprise servers

### Access Logs

- Track who views which documents
- See when documents were accessed
- Audit trail for compliance
